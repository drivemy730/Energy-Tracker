# Energy Tracker 🌍

**Energy Tracker** is a platform that stores and visualizes real data from the **ONU (United Nations Organization)** on how renewable energies are produced and consumed. 🌞💧💨

---

## Data Sources 📊

The project uses data from the ONU database spanning **1965–2022**, including:

- **Hydro** 💧  
- **Solar** ☀️  
- **Wind** 🌬️  
- **Geothermal** 🌋  

The data structure is optimized for efficient management from **1990 to 2022**.

---

## Frontend 🖥️

- Built with **React** for interactive data visualization.

## Backend ⚙️

- Developed with **Java** and **Spring Boot** using REST APIs.

## Database 🗄️

- **PostgreSQL** for reliable storage and querying of energy data.

---

## Features 🎯

- Explore historical renewable energy production and consumption trends.  
- Compare energy sources over time.  
- Interactive charts and dashboards for analysis.
